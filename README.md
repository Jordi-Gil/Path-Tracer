# Path Tracing

Wellcome to Path Tracing Project!

### Example of usage

To see the help
```sh
./path_tracing_sec -h
```
To run the default version
```sh
./path_tracing_sec -d
```

### Clean and Compile

```sh
make clean
make all -j
```
